HTML:

A lot of people have no idea what HTML is. They don't know that it is basically the backbone of internet web pages. On this page, I am going to explain the basics of the HTML language.

1. What is HTML and why is it important?

HTML (Hyper-text Mark-up Language) is the backbone of web pages. It is used in conjunction with CSS to make the page you see. It is used to formtat the text, layout, images, links, everthing you see on the page. The combination of these three elements has come to be known as XHTML. Without HTML there would be no websites or pages.

2. How is it used?

A website author inserts HTML codes into a plain text file to format the content. The author inserts HTML tags, or commands, before and after words to indicate their format and location on the page. HTML tags are also used to add tables, lists, images, music, and other elements to a webpage.

3. Where did it come from?

Tim Burners-Lee defined the URL, HTTP, and HTML specifications regarding the internet at CERN.

4. How is it written?

HTML is written in the form of HTML elements consisting of tags enclosed in angle brackets (like < html >, only without the spaces between the brackets and the tags). HTML tags most commonly come in pairs like < h1 > and < /h1 >, although some tags represent empty elements and so are unpaired, for example < img >. Tags help you format your website, by starting a new paragraph, making a title in a different font, and even adding links. Please keep in mind that if you would write any tags, you would need to make sure that you are NOT adding any spaces between the brackets and the tags. The reason they are like this is that if I didn't have any spaces, the tags would do their actual job. A list of tags will be below.

5. What is CSS and why is it important?

CSS is an extremely powerful style sheet language which is used to control the look and feel of the content written in HTML. So what does this mean? It means that if we have a website element, let's say a "title", and we want to make the text size "20px", the color "black", and the padding around it "10px", we would use CSS to control the look of our element.

6. How do tags work?

As I said before, everything you do in a web page is using tags. The basic format of a tag is this:

< TAG > (something) < /TAG >

7. What is a tag?

HTML tags are keywords (tag names) surrounded by angle brackets. They are explained in question number four.

8. How do you write one?

You can write a tag by pressing the open angle bracket key (press and hold shift and then press the , button), then your keyword (such as p for a new paragraph), then the close angle bracket (shift plus . button). Some keywords will be listed below. If you type in a gibberish keyword in between the brackets (such as 12 15 15 13 9 14 1 18 20 25 18 5 22 5 1 12 5 4), nothing will happen and the keyword will not show up on your webpage.

9. What are attributes and how do they work?

HTML elements have attributes Attributes provide additional information about an element Attributes are always specified in the start tag Attributes come in name and value pairs like: name="value"

10. How do you change attributes?

"< !DOCTYPE html > < html >

< head >

< style >

body { background-color:lightgrey }

h1 { color:blue }

p { color:green }

< /style >

< /head >

< body >

< h1 >This is a heading< /h1 >

< p >This is a paragraph.< /p >

< /body >

< /html >"

This would change a simple title and text from:

This is a paragraph

To:

11. What are some examples?

text() - Sets or returns the text content of selected elements

html() - Sets or returns the content of selected elements (including HTML markup)

val() - Sets or returns the value of form fields

12. Descriptions of some tags:

< p > This is to start a new paragraph.

< body > This is to start a new body.

< font color="green" > This changes the text after the tag green until you write < /font > to make the text back to default.

< img src="putfilenamehere.jpg" alt "mountain view" > This inputs an image to your webpage.
